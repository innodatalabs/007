Welcome to the Agent 007 Documentation

Your guide to setting up and using Agent 007 with Dropbox and OpenAI

Agent 007 allows you to interact with an LLM that has access to a scoped part of your Dropbox. It can create, read, and list files.

You will need both Dropbox and OpenAI accounts to use this service.

Preparation

  • Obtain an OpenAI API key
  • Configure the agent to use your OpenAI API key
  • Create a new application within your Dropbox and configure it for use with the agent
  • Configure the agent to use your Dropbox application

LLM Configuration

Our goal is to obtain an OpenAI API key and configure the agent to use this key.

Steps:

  1. Go to platform.openai.com and sign up for an account or log in if you already have one.
  2. In the top-right corner of the dashboard, click on your profile icon or a similar link to navigate to the API section.
  3. If you don't have an existing project, you may need to create a new one first.
  4. On the API Keys page, select the option to "Create new secret key".
  5. Provide a name for your API key.
  6. Copy and secure: A new secret key will be generated. Copy it immediately and store it in a secure, private location.
  7. Go to agent settings and select OpenAI as agent's engine, then paste the key into the API Key field.
  8. Congratulations! Get yourself some coffee.

Note: Keep your API key secure and never share it publicly.

Dropbox Configuration

Now we will create an app in your Dropbox account and configure it.

Steps:

  1. Go to dropbox.com and log into your account.
  2. On the left panel, navigate to "More" and "App Center". Alternatively, go directly to dropbox.com/apps.
  3. In the "Dropbox App Center", click on the "Build an app" link under the "Manage" header in the left column. Alternatively, go directly to dropbox.com/developers.
  4. Click on the "Create Apps" button.
    • Choose an API: select "Scoped access".
    • Choose the type of access you need: select "App folder". Do not allow this app to have unlimited access to your full Dropbox!
    • Name your app. You can choose anything you like, such as "Playground" or "Write assist".
    • Click the "Create App" button at the bottom of the form.
    You will be redirected to the app configuration page.
  5. Now let's configure your new app:
    • In the "OAuth 2" section, enter the following into the "Redirect URIs" field (copy and paste): https://007.innodatalabs.net/auth/callback and click the "Add" button.
    • Copy the "App key" value and paste it into the Agent configuration "APP_KEY" field.
    • In the "App secret" section, copy the value and paste it into the Agent configuration "APP_SECRET" field.
    • Go to the "Permissions" tab (select the tab at the top) and enable the following permissions:
      • files.metadata.read
      • files.content.write
      • files.content.read
      Click "Submit" to persist your changes.
  6. We're almost there! Return to the agent settings page and click the "Connect" button at the bottom. This will connect the agent to your new Dropbox application.

Important: Do not grant unlimited access to your full Dropbox. The app folder access is more secure.

Perfect! This completes the configuration. Enjoy using Agent 007!
